API Queries

Users can access data via a RESTful API with several different endpoints.

Query variables must be encoded. Commas are encoded as "%2c" and spaces are encoded as "%20". For example, to query ships per day in San Francisco, CA:

` /smokestackAPI/v1.0/port_query_shipsPerDay/San%20Francisco%2c%20CA `

Responses are returned in JSON format.

For more info, see: Smoke Stack Github Repo

Query Variable Return Value URL
Port timeseries: ships per day '/smokestackAPI/v1.0/port_query_shipsPerDay/"port"'
Port timeseries: total visit time per quarter '/smokestackAPI/v1.0/port_query_visitTimeQuarterly/"port"'
Ship series: total number of visits per port '/smokestackAPI/v1.0/ship_query_visitsPerPort/"ship"'
Ship series: total visit time per port '/smokestackAPI/v1.0/ship_query_totalTimePerPort/"ship"'
